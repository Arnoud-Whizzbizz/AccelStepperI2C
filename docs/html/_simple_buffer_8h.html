Detailed Description

simple and ugly serialization buffer for any types. Template technique adapted from Nick Gammon's I2C_Anything library, thanks, Nick!

Simple and ugly serialization buffer for any data type. Just a buffer, a current position pointer used for reading from and writing to the buffer, and template functions for reading and writing any data type. The template technique is adapted from Nick Gammon's I2C_Anything library.
